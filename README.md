# InventoryApp
 A mobile app to take inventory of the house's pantry
 Is made with React Native

## Setup

1. Install Expo

    `npm install -g expo-cli`

2. Install some dependencies

    ```
    npm install @react-navigation/stack
    npm install @react-navigation/native
    ```

2. Start the project

    `npm start`

    Or

    `npm start --android`
